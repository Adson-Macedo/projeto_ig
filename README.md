## Pré-requisitos
- [Node 12.16.1+](https://nodejs.org/en/download/)

## Instalação e Execução
#### 1. Instalar dependências
```sh  
$ npm install    
```
 
#### 2. Executar o projeto
```sh  
$ npm start
```
Navegar para http://localhost:3000
